# Legal Policies

Centralized Terms of Service and Privacy Policy for apps and services operated by **ScTru**.

## Usage

Link to these documents from any app:

- **Terms of Service:** `https://sctru.github.io/legal-policies/terms-of-service`
- **Privacy Policy:** `https://sctru.github.io/legal-policies/privacy-policy`

For app-specific policies, add a subdirectory (e.g., `foamfit/privacy-policy.md`) and link to:
`https://sctru.github.io/legal-policies/foamfit/privacy-policy`

## Structure

```
├── terms-of-service.md     # Default ToS (all apps)
├── privacy-policy.md       # Default Privacy Policy (all apps)
└── <app-name>/             # App-specific overrides (optional)
    ├── terms-of-service.md
    └── privacy-policy.md
```
